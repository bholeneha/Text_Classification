Latent Dirichlet Allocation ("LDA")

LDA identifies the hidden topic structures in text documents. Using Bayesian statistics and Dirichletdistributions to process and identify topics. In order to use LDA data must first be pre-processed through normalization, stemming, stop word removal and tokenization. The model lda_model = gensim.models.LdaMulticore(corpus, id2word=id2word,num_topics) was used to build the model
